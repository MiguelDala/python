Exercícios de Python

O que é Python?

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. É conhecida por sua sintaxe clara e legível, sendo amplamente utilizada em desenvolvimento web, automação, ciência de dados, inteligência artificial e muito mais.

História do Python

Python foi criado no final dos anos 1980 por Guido van Rossum e lançado em 1991. A linguagem foi projetada com a filosofia de tornar a programação mais acessível e eficiente, enfatizando a simplicidade e a legibilidade do código.

a. Verificar se um número é positivo ou negativo

numero = int(input("Digite um número: "))
if numero > 0:
    print("O número é positivo.")
elif numero < 0:
    print("O número é negativo.")
else:
    print("O número é zero.")

b. Verificar se um número é par ou ímpar

numero = int(input("Digite um número: "))
if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

c. Verificar se um número está dentro de um intervalo

numero = int(input("Digite um número: "))
if 1 <= numero <= 10:
    print("O número está entre 1 e 10.")
else:
    print("O número não está entre 1 e 10.")

d. Verificar se uma pessoa é maior de idade

idade = int(input("Digite sua idade: "))
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

e. Classificar a nota do aluno

nota = float(input("Digite a nota do aluno: "))
if nota >= 7:
    print("Aprovado")
elif 5 <= nota < 7:
    print("Recuperação")
else:
    print("Reprovado")

a. Soma dos elementos de uma lista

numeros = [1, 2, 3, 4, 5]

b. Fatorial de um número

numero = 5
fatorial = 1
for i in range(1, numero + 1):
    fatorial *= i
print("Fatorial de", numero, "é:", fatorial)

c. Iterar sobre caracteres de uma string

for caractere in texto:
    print(caractere)

d. Números pares entre 1 e 20

a. Soma dos elementos de uma lista utilizando while

numeros = [1, 2, 3, 4, 5]
soma = 0
i = 0
while i < len(numeros):
    soma += numeros[i]
    i += 1
print("Soma dos números:", soma)

b. Fatorial de um número utilizando while

numero = 5
fatorial = 1
i = 1
while i <= numero:
    fatorial *= i
    i += 1
print("Fatorial de", numero, "é:", fatorial)

c. Iterar sobre caracteres de uma string utilizando while

texto = "Python"
i = 0
while i < len(texto):
    print(texto[i])
    i += 1

d. Números pares entre 1 e 20 utilizando while

num = 1
while num <= 20:
    if num % 2 == 0:
        print(num)
    num += 1

e. Lista de quadrados dos números de 1 a 10 utilizando while

quadrados = []
x = 1
while x <= 10:
    quadrados.append(x**2)
    x += 1
